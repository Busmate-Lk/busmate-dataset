# Thibus Font to Unicode Sinhala Converter

This used `https://ucsc.cmb.ac.lk/ltrl/services/feconverter/?maps=s_tbs-u.xml` online converter to convert thibus font used content into unicoded content.


- "new-approach/all-pdf-content.txt" is the file that cntains all raw pdf content.

- "new-approach/all-converted-content.txt" is the file that contains all converted content using `https://ucsc.cmb.ac.lk/ltrl/services/feconverter/?maps=s_tbs-u.xml` online converter.
